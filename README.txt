

Sistema de Estoque em C

Um sistema simples de controle de estoque desenvolvido em linguagem C, com arquitetura modular e uso de struct para gerenciamento de produtos.
O projeto foi criado para treinar conceitos que aprendi no curso de algaritimos e logica, também aprendi funções e modularização.

---

Funcionalidades

* Adicionar produtos
* Listar produtos cadastrados
* Buscar produto pelo ID
* Atualizar quantidade
* Remover produto
* Mostrar valor total do estoque
* Listar produto com estoque baixo

---

Estrutura do Projeto

```
📁 projeto-estoque
│-- 📄 master_principal.c          # Função principal do programa
│-- 📄 produto.c       # Implementação das funções relacionadas a produtos
│-- 📄 produto.h       # Definições e estruturas do produto
│-- 📄 util.c          # (Opcional) Funções auxiliares
│-- 📄 util.h          # (Opcional) Cabeçalho das funções auxiliares
```
